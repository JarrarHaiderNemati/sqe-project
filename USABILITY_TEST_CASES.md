# Swift Response - Usability Test Cases

Usability testing is normally performed manually using heuristic evaluation and real-user observation. The SQE document lists this as:

```text
Manual + heuristic evaluation
```

## Usability Heuristics Used

- Visibility of important actions
- Clarity of labels and text
- Ease of navigation
- Error prevention
- Mobile friendliness
- Consistency of design
